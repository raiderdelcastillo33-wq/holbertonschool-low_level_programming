# C - Simple Calculator

## Compile

gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator

## Run

./calculator

## Supported operations

Task 0 displays the calculator menu and allows quitting with option 0.

## Numeric behavior

This task reads an integer menu option using scanf.

## Known limitations

Task 0 does not perform arithmetic operations yet.
Non-numeric input is not handled in the mandatory scope.
