# C - Function pointers

This project covers the use of function pointers in C.

## Learning objectives

- Understand what function pointers are.
- Understand how to declare and use function pointers.
- Understand how to use functions passed as arguments to other functions.

## Files

- `0-print_name.c` - Prints a name using a callback function.
- `function_pointers.h` - Contains the function prototypes for the project.
