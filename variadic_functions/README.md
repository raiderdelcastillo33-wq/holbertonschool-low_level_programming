# C - Variadic functions

This project covers variadic functions in C.

## Learning objectives

- Understand what variadic functions are.
- Use `va_start`, `va_arg`, and `va_end`.
- Understand the `const` type qualifier.

## Files

- `0-sum_them_all.c` - Returns the sum of all variadic arguments.
- `variadic_functions.h` - Contains the function prototypes for the project.
